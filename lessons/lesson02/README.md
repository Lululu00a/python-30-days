# 课时 02 示例代码

| 文件 | 说明 |
|------|------|
| `types_demo.py`     | 5 大基础类型演示（int/float/bool/str/None） |
| `variable_label.py` | "变量是标签"的可变/不可变对象对比 |
| `fstring_demo.py`   | f-string 格式化全攻略（含调试技巧） |

## 运行
```powershell
python lessons/lesson02/types_demo.py
python lessons/lesson02/variable_label.py
python lessons/lesson02/fstring_demo.py
```

## 学习建议
1. 不要只是看，**亲自运行每一个示例**
2. 修改示例中的值，观察输出变化
3. 在 REPL 里 `python` 启动后逐行复现，加深理解
