## 练习 2：is vs ==（验证小整数缓存）

创建 `exercise2_int_cache.py`，编写代码验证：
1. `a = 256; b = 256; a is b` 的结果
2. `a = 257; b = 257; a is b` 的结果
3. `a = -5; b = -5; a is b` 的结果
4. `a = -6; b = -6; a is b` 的结果（提示：缓存范围是 -5~256）

输出格式：
```
a=256, b=256, a is b? True
a=257, b=257, a is b? False
...
结论：Python 缓存了 -5 到 256 的小整数
```

## 练习 5：BMI 计算器（综合应用）

创建 `exercise5_bmi.py`，写一个简易 BMI 计算器：

1. 用 `input()` 获取用户身高（米）和体重（千克）
2. 计算 BMI = 体重 / 身高²
3. 根据 BMI 给出建议：
   - < 18.5 → "偏瘦"
   - 18.5 ~ 23.9 → "正常"
   - 24 ~ 27.9 → "偏胖"
   - ≥ 28 → "肥胖"
4. 用 f-string 输出，BMI 保留 1 位小数

预期交互：
```
请输入你的身高（米）：1.75
请输入你的体重（千克）：70
你的 BMI 是 22.9，体型评估：正常
```

⚠️ 注意：`input()` 返回 **str**，需要用 `float()` 转换！
