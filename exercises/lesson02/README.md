## 练习 4：可变 vs 不可变（理解题）

阅读以下代码，**先猜输出**，再运行验证：

```python
a = "hello"
b = a
b += " world"
print(f"a = {a}")
print(f"b = {b}")

x = [1, 2, 3]
y = x
y.append(4)
print(f"x = {x}")
print(f"y = {y}")
```

把你的答案和解释写到 `solutions/lesson02/exercise4_thinking.md`。

## 练习 5：BMI 计算器

输入身高（米）、体重（千克），计算 BMI 并判断：

| BMI 范围 | 结论 |
|----------|------|
| < 18.5 | 偏瘦 |
| 18.5 ~ 24 | 正常 |
| 24 ~ 28 | 偏胖 |
| ≥ 28 | 肥胖 |

输出示例：

```
请输入身高（米）：1.75
请输入体重（千克）：70
你的 BMI 是 22.86，结论：正常
```

> 提示：BMI 公式：体重 / 身高²
