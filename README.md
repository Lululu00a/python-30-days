# Python 30 天学习计划

> 面向**有 C 语言基础**、想做 Python 项目的学员
> 30 课时（每课 25–30 分钟）后达到**初级 Python 工程师**水平
> 能够独立阅读 Flask / FastAPI / 爬虫等中型开源项目

---

## 🚀 快速开始

```powershell
# 1. 创建虚拟环境
python -m venv .venv

# 2. 激活（Windows PowerShell）
.venv\Scripts\Activate.ps1
# Windows CMD:    .venv\Scripts\activate.bat
# macOS / Linux:  source .venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行第一个程序
python lessons/lesson01/hello.py
```

## 💡 学习建议

1. **每天 1 课时**，避免遗忘曲线影响
2. **从 C 思维过渡**：每学一个概念，主动对比 C 的做法
3. **代码量底线**：每课时手敲 ≥ 100 行
4. **善用 REPL**：随时验证想法
5. **每天提交 Git**：养成工程化习惯
