# 练习 4 思考题答案

## 输出结果

```
a = hello
b = hello world

x = [1, 2, 3, 4]
y = [1, 2, 3, 4]
```

## 解释

### 字符串部分（不可变）

- `a = "hello"`：标签 a 贴在字符串对象 "hello" 上
- `b = a`：标签 b 也贴在同一个 "hello" 上
- `b += " world"`：**字符串不可变**，所以这里实际是创建了一个新字符串 "hello world"，然后把标签 b 撕下来贴到新对象上
- 标签 a 仍然贴在原来的 "hello" 上 → a 没变

### 列表部分（可变）

- `x = [1, 2, 3]`：标签 x 贴在 list 对象上
- `y = x`：标签 y 也贴在同一个 list 上（**两个标签同一个对象**）
- `y.append(4)`：**列表可变**，直接修改对象本身
- 标签 x 和 y 指向同一个对象，所以**都看到了变化**

## 核心结论

| 操作 | 不可变对象（str/int/tuple） | 可变对象（list/dict/set） |
|------|-----|-----|
| `b = a` | 共享对象 | 共享对象 |
| 修改 b | **创建新对象**，不影响 a | **原地修改**，影响所有引用 |

> 提示：想"复制" list 用 `y = x.copy()` 或 `y = x[:]`
